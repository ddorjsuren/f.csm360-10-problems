1. RGB7010 - 3 оронтой тоо

2. RGB7012 - Цаг минут секунд
Өгөгдсөн секундыг цаг минут секунд болго.

3. RGB7022 - Арифметик прогрессын N-р гишүүн
Бүхэл тоон арифметик прогрессын p дэх гишүүн Ap болон q дэх гишүүн Aq өгөгдсөн бол N дүгээр гишүүнийг ол.

4. RGB7031 - Гурвалжны талбай
Гурвалжны гурван тал өгөгдсөн бол талбайг нь ол.

5. RGB7063 - Тоонд харгалзах тэмдэгт
Өгөгдсөн тоонд харгалзах ASCII тэмдэгтийг хэвлэ.

6. RGB7072 - Нийлбэр 3
Өгөгдсөн сондгой тоо n хүртэлх натурал сондгой тоонуудын нийлбэрийг ол. Давталт ашиглахгүй.

7. RGB7092 - Товчлуурууд
Манао нэлээд чадвар шаардсан цоож онгойлгохыг оролдож байна. Цоож n товчлуураар ба онгойлгохын тулд тогтмол дарааллаар товчлуурыг дарах хэрэгтэй. Товч дарах үед хэрэв зөв товчлуураа дарсан бол дараастай чигээр үлддэг ба эсрэг тохиолдолд бүх дарагдсан товчлуурууд анхны төлөвтөө ордог байна. 

8. RGB7109 - Тэнцүү 5
Өгөгдсөн 3 тоон дотроос 5-тай тэнцүү тоонуудын тоог ол.

9. RGB7118 - Гараг
Долоо хоногийн дугаар өгөгдөхөд харгалзах гарагийг англиар хэвлэ. Эхний үсгийг л томоор бичнэ.

10. RGB7127 - Саваануудтай тоглоом
n ширхэг хөндлөн, m ширхэг босоо саваагаар хийсэн торон дээр тоглоом тоглоно.Огтлолцлын цэг гэж хөндлөн болон босоо савааны огтлол дээр үүссэн ямар нэг цэгийг хэлэх юм.Тоглогчид ээлжлэн нүүх ба Акшат эхлээд нүүнэ. Нүүж байхдаа тоглогч ямар нэг үлдсэн огтлолцлын цэгийг сонгох ба энэ цэгийг дайран өнгөрөх бүх савааг торноос арилгана. Цааш нүүж чадахгүй болсон тоглогч ялагдана. Өөрөөр хэлбэл түүнийг нүүхэд(turn) торонд ямар ч огтлолцлын цэг байхгүй бол.Хоёр тоглогч байж болох хамгийн зөв аргаар тоглосон гэж бодъё. Хэн нь хожих вэ?
